# Tipo de datos primitivos
## Objetivo del módulo
- Reconocer los 8 tipos de datos primitivos
- Entender coerción implicita
- Diferenciar == vs ===
- Detectar errores comunes
- Convertir tipos manualmente
- Conocer un poco mas de javascript (Operadores y Cadenas)

### Los 8 tipos de datos primitivos
| Tipo          | Ejemplo         | Nota                              |
| ------------- | --------------- | --------------------------------- |
| string        | `"hola"`        | Texto                             |
| number        | `10`, `3.14`    | Todo es number (no hay int/float) |
| boolean       | `true`, `false` | Verdadero/falsp                   |
| undefined     | `var a;`        | Declarado pero no iniciado        |
| null          | `null`          | vacio                             |
| bigint        | `10n`           | Número muy grandes                |
| symbol        | `Symbol("id")`  | Identificadores únicos            |
| **extra** NaN | `0/0`           | ¨Not A Number¨                    |
