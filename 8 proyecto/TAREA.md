# Completar el proyecto
## 1. Marcat tarea como completado
- Agregar el botón "Completar"
- Distinguir el completado

## 2. Filtro de tareas
Filtrar mediante una lista
- Mostrar todas las tareas
- Mostrar solo pendientes
- Mostrar solo completados

## 3. Ordenar alfabéticamente
Botón: Ordenar tareas de la A-Z

## 4. Contador
Mostrar:
- Total: x
- Completadas: y
- Pendientes: z

## 5. Agregar de fecha de creación
Al crear insertar la fecha
```js
new Date()
```
Mostrar en la lista

# Desplegar

1. Ingresa a: https://www.netlify.com/
2. Suben el proyecto
3. Van a Project details y change project name
   1. Colocan su nombre completo
4. Obtienen el dominio
